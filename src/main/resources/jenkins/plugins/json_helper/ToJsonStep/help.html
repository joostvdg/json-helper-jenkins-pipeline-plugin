Converts a JSON String to an Object.
You can supply it an object and it will write out a correct json, or return null.

If you want to create an object from scratch, you can use the following: 

        def user = new Expando(username: 'mrhaki')
        def jsonString = writeObjectToJson(user)
        echo "jsonString=$jsonString"
It will print out the following: 
        jsonString={"expandoProperties":{"username":"mrhaki"}}

For more about the Expando object, read: http://mrhaki.blogspot.nl/2009/10/groovy-goodness-expando-as-dynamic-bean.html
